# Projeto com CI 

Este projeto demonstra a configuração de um ambiente de testes utilizando `pytest` e `pytest-cov`, integrado com uma esteira de Integração Contínua (CI) via GitHub Actions.

---

## Como instalar as dependências

Certifique-se de ter o Python instalado em sua máquina. Em seguida, clone o repositório, navegue até a pasta raiz e execute o comando abaixo para instalar as ferramentas necessárias:

```bash
pip install -r requirements.txt
